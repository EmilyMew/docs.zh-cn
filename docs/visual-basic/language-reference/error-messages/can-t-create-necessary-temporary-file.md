# <a name="cant-create-necessary-temporary-file"></a>无法建立必要的临时文件。
其中一个驱动器已满，其中包含由 TEMP 环境变量指定的目录或 TEMP 环境变量指定的无效或只读的驱动器或目录。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1.  如果完全从驱动器中，删除文件。  
  
2.  在 TEMP 环境变量中指定一个不同的驱动器。  
  
3.  指定有效的驱动器 TEMP 环境变量。  
  
4.  从当前指定的驱动器或目录中删除只读的限制。
