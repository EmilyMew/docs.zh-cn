# <a name="constant-expression-not-representable-in-type-typename"></a>常量表达式无法在类型中表示\<类型名称 >
尝试评估一个常量，它不能适应目标类型，通常是因为溢出范围。  
  
 **错误 ID:** BC30439  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1.  目标类型更改为一个可以处理该常量。
