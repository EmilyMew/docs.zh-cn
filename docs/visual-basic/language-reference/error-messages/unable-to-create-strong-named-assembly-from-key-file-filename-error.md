# <a name="unable-to-create-strong-named-assembly-from-key-file-filename-error"></a>无法从密钥文件创建强名称程序集 '\<文件名 >:\<错误 >
无法从指定的密钥文件中创建的强名称程序集。  
  
 **错误 ID:** BC31026  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1.  验证已指定正确的密钥文件，并且它未锁定由其他应用程序。
