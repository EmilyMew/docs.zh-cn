# <a name="type-mismatch-visual-basic"></a>类型不匹配 (Visual Basic)
你试图将值转换为另一种类型不是有效的方式。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1.  检查以确保它是有效的分配。  
  
2.  确保未将对象传递给过程所需的单个属性或值。  
  
3.  请确保你未使用的模块或项目名称应为表达式。
