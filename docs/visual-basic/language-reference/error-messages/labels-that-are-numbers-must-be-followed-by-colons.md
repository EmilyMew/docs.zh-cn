# <a name="labels-that-are-numbers-must-be-followed-by-colons"></a>数字标签后面必须跟冒号
行号遵循与其他类型的标签，相同的规则，并且必须包含一个冒号。  
  
 **错误 ID:** BC30801  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   位置号后跟一个冒号开头的一行代码;例如：  
  
    ```  
    400:    X += 1  
    ```
