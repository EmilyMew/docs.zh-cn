# <a name="protocolstopfailure"></a>ProtocolStopFailure
ID:147  
  
 严重性：Error  
  
 类别：TransactionBridge  
  
## <a name="description"></a>描述  
 此事件指示 WS-Atomic Transaction 协议服务无法停止。 此事件将列出协议 ID、协议名称、异常、进程名称和进程 ID。
