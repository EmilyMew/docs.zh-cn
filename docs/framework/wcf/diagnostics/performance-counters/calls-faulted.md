# <a name="calls-faulted"></a>Calls Faulted（出错的调用次数）
计数器名称：Calls Faulted（出错的调用次数）  
  
## <a name="description"></a>描述  
 向此操作返回的出错的调用次数。 在 Windows Communication Foundation (WCF) 应用程序中服务方法进行通信使用 SOAP 错误消息的处理错误信息。 SOAP 错误是包括在服务操作元数据中的消息类型，因此会创建一个错误协定，客户端可使用该协定来使执行更加可靠或更具交互性。 由于 SOAP 错误在客户端以 XML 格式表示，因此具有高度的互操作性。
