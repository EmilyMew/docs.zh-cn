# <a name="corimageunloading-function"></a>_CorImageUnloading 函数
卸载托管的模块映像时通知加载程序。  
  
 未实现此函数。 如果调用，则返回 E_NOTIMPL。  
  
## <a name="syntax"></a>语法  
  
```  
STDAPI (VOID) _CorImageUnloading(   
   [in] PVOID* ImageBase  
);  
```  
  
## <a name="parameters"></a>参数  
 `ImageBase`  
 [in]指向要卸载的图像的起始位置的指针。
