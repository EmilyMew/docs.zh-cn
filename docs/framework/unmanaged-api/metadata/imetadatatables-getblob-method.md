# <a name="imetadatatablesgetblob-method"></a>IMetaDataTables::GetBlob 方法
获取一个指向二进制大型对象 (BLOB) 中的指定的列索引处。  
  
## <a name="syntax"></a>语法  
  
```  
HRESULT GetBlob (  
    [in]  ULONG          ixBlob,  
    [out] ULONG          *pcbData,  
    [out] const void     **ppData  
);  
```  
  
## <a name="parameters"></a>参数  
 `ixBlob`  
 [in]要从其中获取的内存地址`ppData`。  
  
 `pcbData`  
 [out]指向的大小，以字节为单位的`ppData`。  
  
 `ppData`  
 [out]检索到的二进制数据的指针的指针。
