# <a name="imetadataemitdeleteclasslayout-method"></a>IMetaDataEmit::DeleteClassLayout 方法
销毁指定的标记所表示的类型的类布局的元数据签名。  
  
## <a name="syntax"></a>语法  
  
```  
HRESULT DeleteClassLayout (  
    [in]  mdTypeDef   td  
);  
```  
  
## <a name="parameters"></a>参数  
 `td`  
 [in]`mdTypeDef`元数据标记所表示的类型将为其删除的类布局。
