# <a name="imetadatatablesgetnumtables-method"></a>IMetaDataTables::GetNumTables 方法
获取在当前作用域中的表数`IMetaDataTables`实例。  
  
## <a name="syntax"></a>语法  
  
```  
HRESULT GetNumTables (  
    [out]  ULONG   *pcTables  
);  
```  
  
## <a name="parameters"></a>参数  
 `pcTables`  
 [out]指向当前实例作用域中的表数的指针。
