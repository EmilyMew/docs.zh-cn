# <a name="imetadatatablesgetnextstring-method"></a>IMetaDataTables::GetNextString 方法
获取当前表的列中的下一个字符串的索引。  
  
## <a name="syntax"></a>语法  
  
```  
HRESULT GetNextString (   
    [in]  ULONG   ixString,  
    [out] ULONG   *pNext  
);  
```  
  
## <a name="parameters"></a>参数  
 `ixString`  
 [in]从字符串表的列索引值。  
  
 `pNext`  
 [out]索引列中的下一个字符串的指针。
