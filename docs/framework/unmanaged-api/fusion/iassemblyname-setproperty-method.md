# <a name="iassemblynamesetproperty-method"></a>IAssemblyName::SetProperty 方法
设置引用的指定的属性标识符的属性的值。  
  
## <a name="syntax"></a>语法  
  
```  
HRESULT SetProperty (  
    [in] DWORD  PropertyId,  
    [in] LPVOID pvProperty,  
    [in] DWORD  cbProperty  
);  
```  
  
## <a name="parameters"></a>参数  
 `PropertyId`  
 [in]将设置其值的属性的唯一标识符。  
  
 `pvProperty`  
 [in]要设置所引用的属性值`PropertyId`。  
  
 `cbProperty`  
 [in]大小，以字节为单位的`pvProperty`。
