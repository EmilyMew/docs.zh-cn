# <a name="iassemblynamegetproperty-method"></a>IAssemblyName::GetProperty 方法
获取一个指针指向引用指定的属性标识符的属性。  
  
## <a name="syntax"></a>语法  
  
```  
HRESULT GetProperty (  
    [in]      DWORD    PropertyId,  
    [out]     LPVOID   pvProperty,  
    [in, out] LPDWORD  pcbProperty  
);  
```  
  
## <a name="parameters"></a>参数  
 `PropertyId`  
 [in]请求的属性的唯一标识符。  
  
 `pvProperty`  
 [out]返回的属性的数据。  
  
 `pcbProperty`  
 [in、 out]大小，以字节为单位的`pvProperty`。
