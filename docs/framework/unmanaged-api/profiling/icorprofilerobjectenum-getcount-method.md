# <a name="icorprofilerobjectenumgetcount-method"></a>ICorProfilerObjectEnum::GetCount 方法
获取集合中的冻结对象总数。  
  
## <a name="syntax"></a>语法  
  
```  
HRESULT GetCount (  
    [out] ULONG   *pcelt  
);  
```  
  
## <a name="parameters"></a>参数  
 `pcelt`  
 [out]指向集合中的冻结对象数的指针。  
  
 此方法将始终返回零的.NET Framework 3.5 版 Service Pack 1 (SP1) 和更高版本。
