# <a name="isymunmanagedreader2getsymattributepreremap-method"></a>ISymUnmanagedReader2::GetSymAttributePreRemap 方法
获取根据其名称的自定义属性。 与不同的元数据自定义特性，这些属性保存在符号存储区中。  
  
## <a name="syntax"></a>语法  
  
```  
HRESULT GetSymAttributePreRemap(  
    [in]  mdToken  parent,  
    [in]  WCHAR    *name,  
    [in]  ULONG32  cBuffer,  
    [out] ULONG32  *pcBuffer,  
    [out, size_is(cBuffer),  
        length_is(*pcBuffer)] BYTE buffer[]);  
```  
  
## <a name="parameters"></a>参数  
 `parent`  
 [in]父元数据标记。  
  
 `name`  
 [in]一个指向`WCHAR`，其中包含名称。  
  
 `cBuffer`  
 [in]一个`ULONG32`指示的大小`buffer`数组。  
  
 `pcBuffer`  
 [out]一个指向`ULONG32`接收包含属性字节所需的缓冲区的大小。  
  
 `buffer`  
 [out]指向接收属性字节的缓冲区的指针。  
  
## <a name="return-value"></a>返回值  
 如果方法成功，则为 S_OK否则为 E_FAIL 或某些其他错误代码。  
  
## <a name="requirements"></a>要求  
 **标头：** CorSym.idl CorSym.h
