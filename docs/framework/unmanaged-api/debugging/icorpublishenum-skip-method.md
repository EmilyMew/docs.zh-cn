# <a name="icorpublishenumskip-method"></a>ICorPublishEnum::Skip 方法
将光标向前移动在枚举中指定数目的项。  
  
## <a name="syntax"></a>语法  
  
```  
HRESULT Skip (  
    [in] ULONG   celt  
);  
```  
  
## <a name="parameters"></a>参数  
 `celt`  
 [in]要向前移动游标的项的数目。
